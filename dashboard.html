About This Page

This is the Dashboard of Valorant Insights+. After signing in, users land here to see an overview of their stats, including K/D ratio, win rate, headshot accuracy and their recent matches.

The "Analyze" button is the key component of the website. Using my knowledge of Valorant, data analysis and with the help of the Valorant Api to access data such as matches, locations of players during key events (kills and plants). I want to provide thoughtful and personalized insights to the user, which can help improve their future gameplay.

For instance, below you can see a picture of haven map death locations of a user. Looking at the map we can see that, he has died multiple times in A lobby. With further analysis we can check the angles the user is struggling at in this location and provide tips to improve their performance.
